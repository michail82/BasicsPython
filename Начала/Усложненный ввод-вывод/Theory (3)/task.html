Есть задачи, в которых вводимые данные заданы в одной строке.

Чтобы была возможность работать с такими данными, их надо разделить между переменными.

Для этого нужно применить следующий синтаксис:

a, b = input().split()

Здесь есть правило: количество переменных должно соответствовать количеству данных. В примере выше в одной строке передаются два целых числа, разделенных пробелами. Соответственно, нужно эти данные передать также двум переменным. Осталось не забыть, что определенные таким образом переменные имеют строковый тип. Для арифметического сложения их нужно привести к типу целого числа.

a = int(a)
b = int(b)

или одной строкой

a, b = int(a), int(b)

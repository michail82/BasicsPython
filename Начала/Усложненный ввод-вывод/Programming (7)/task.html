Напишите программу, которая выдает строку, получаемую путём повторения N раз. Данные N и текстовая цепочка символов вводятся на разных строках. Вспоминаем про умножение строки на целое число.
Sample Input:
10
wsx

Sample Output:
wsxwsxwsxwsxwsxwsxwsxwsxwsxwsx


Memory limit: 256 MB
Time limit: 15 seconds

Если вывод необходимо организовать на одной строке, то все текстовые выражения и переменные нужно перечислить через запятую в одной функции print():

a = 'Мама'
b = 'мыла'
c = 'раму'
print(a,b,c)  # Выведет Мама мыла раму

тоже и с численными выражениями

a = 10  # длина прямоугольника
b = 15  # ширина прямоугольника
c = (a + b) * 2  # периметр прямоугольника
d = a * b  # площадь прямоугольника
print(с, d)  # Выведет периметр и площадь в одной строке через пробел

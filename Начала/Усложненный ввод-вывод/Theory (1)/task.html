Кроме многострочного ввода может понадобиться вывод на несколько строк. Здесь просто - каждый вызов print() с параметром внутри скобок - вывод новой строки.
Задача. Даны два целых числа на разных строках. Написать программу вывода их суммы и произведения. Вывод осуществить на разных строках.

a = int(input())  # ввод первого числа
b = int(input())  # ввод второго числа
c = a + b  # получение суммы чисел
d = a * b  # получение произведения чисел
print(c)  # вывод результата суммы
print(d)  # вывод результата произведения
или тот же код, только короче: 
a, b = int(input()), int(input())
print(a + b)
print(a * b)

Теперь разберем третий шаг программирования - вывод результатов.

 

Функция print() с аргументом внутри скобок выводит значение этого аргумента в ОДНУ строку на экране.

Примеры:

print(10)  # выведет число 10
print('10')  # выведет строку '10' как текст, только без кавычек
print(ab)  # выведет значение переменной ab
print(ab + 1)  # выведет увеличенное на единицу значение переменной ab
print()  # выведет пустую строку

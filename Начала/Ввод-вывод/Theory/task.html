Теперь перейдём к первому шагу программирования - вводу данных и третьему шагу - выводу результатов. Со вторым шагом мы уже разбирались, когда знакомились с арифметическими операциями над переменными.

 

Функция ввода. Такой функцией является input(). Каждая функция input() захватывает только ОДНУ строку данных и воспринимает ее как ТЕКСТ (строковый тип). Пример, заполнена только одна строка данных в поле для ввода данных на ресурсе rextester.com. Вот эту строку и захватит input().

Но где окажутся эти данные и как мы сможем в дальнейшем с ними работать?

Обычно введенные данные присваивают переменной, чтобы затем их обрабатывать, например: a = input().

Когда мы присвоили переменной значение через функцию input(), мы затем можем легко поменять ей тип на любой другой. Но чаще бывает так, что надо сразу назначить введенным данным тип. Это делается так:

a = input() # введённое значение будет текстом (по умолчанию)
b = int(input()) # введённое значение преобразуется в тип целого числа
c = float(input()) # введённое значение преобразуется в тип вещественного числа
d = str(input()) # введённое значение преобразуется в строковый тип, хотя оно уже текст :)

А дальше нужна обработка и вывод результата.

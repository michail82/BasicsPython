Python 3+ — это современный язык программирования, на котором просто и приятно писать программы. Изучение этого языка позволит участвовать в школьных олимпиадах, сдавать ОГЭ и ЕГЭ.
Почему в названии " 3+", потому, что вышла третья версия этого замечательного языка, которая в достаточной степени отличается от второй версии по синтаксису. Знак плюс означает, что информация из этого курса будет достоверна для всех версий Python, начиная с 3.0.

По большому счету, написание программ на любом языке программирования ﻿сводится к трем шагам:
  1. Получить данные для обработки;
  2. Переработать полученные данные;
  3. Выдать результат обработки.

В этом курсе мы затронем все три шага.

Результат целочисленного деления - это частное от деления нацело, или сколько ЦЕЛЫХ делителей умещается в делимом.

Остаток целочисленного деления - это число, которое получится, если от первоначального числа отнять все целые делители.

Поясним на примере: в семерке умещается две целых тройки и ещё одна единица, то есть 7 = 3 + 3 + 1, значит

7 // 3 = 2

7 % 3 = 1

Взглянем на эти арифметические действия, применительно к целым и вещественным числам:

a1 = 10  # a1 имеет тип целого числа
a2 = 10.1  # a2 имеет тип вещественного числа

b1 = a1 // 2  # b1 имеет тип целого числа. Результат 5
b2 = a2 // 2  # b2 имеет тип вещественного числа. Результат 5.0

с1 = a1 % 2  # с1 имеет тип целого числа. Результат 0
с2 = a2 % 2  # с2 имеет тип вещественного числа. Результат 0.09999999999999964

 

Задачи на применение операций целочисленного деления и остатка целочисленного деления встречаются довольно часто. Нам же важно понимать следующее: если остаток целочисленного деления ЦЕЛОГО числа A на ЦЕЛОЕ число B равен 0, то A кратно B. Таким же образом можно вычислять чётность числа - остаток целочисленного деления числа на 2 должен быть равен 0. Если же остаток не равен 0, то число нечетное.

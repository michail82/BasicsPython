Давайте рассмотрим простые арифметические операции (*, /, +, -) над разными числовыми типами:

a1 = 10  # a1 имеет тип целого числа
b1 = 5.25  # b1 имеет тип вещественного числа

c1 = a1 + a1  # c1 имеет тип целого числа. Результат 20
c2 = b1 + b1  # c2 имеет тип вещественного числа. Результат 10.5 
c3 = a1 + b1  # c3 имеет тип вещественного числа. Результат 15.25

d1 = a1 - a1  # d1 имеет тип целого числа. Результат 0
d2 = b1 - b1  # d2 имеет тип вещественного числа. Результат 0.0 
d3 = a1 - b1  # d3 имеет тип вещественного числа. Результат 4.75
d4 = b1 - a1  # d4 имеет тип вещественного числа. Результат -4.75

e1 = a1 * a1  # e1 имеет тип целого числа. Результат 100
e2 = b1 * b1  # e2 имеет тип вещественного числа. Результат 27.5625
e3 = a1 * b1  # e3 имеет тип вещественного числа. Результат 52.5

f1 = a1 / a1  # f1 имеет тип вещественного числа. Результат 1.0
f2 = b1 / b1  # f2 имеет тип вещественного числа. Результат 1.0
f3 = a1 / b1  # f3 имеет тип вещественного числа. Результат 1.9047619047619047
f4 = b1 / a1  # f4 имеет тип вещественного числа. Результат 0.525

Если с простыми арифметическими операциями (*, /, +, -) всё более-менее понятно, то целочисленное деление (//) и остаток целочисленного деления стоит разобрать подробнее.

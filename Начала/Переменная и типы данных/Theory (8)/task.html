Вот пример задачи, в котором применяется преобразование типов.

Пусть задано число 12. Вычислить значение числа, равное удвоенному трижды повторенному первоначальному числу.

a = 12 # первоначальное число 
b = str(a) # превращаем число в строку, результат '12'
c = b * 3 # трижды повторяем строку, результат '121212'
d = int(c) # превращаем строку в число, результат 121212
e = d * 2 # удваиваем число, результат 242424

Эту же задачу можно решить значительно меньшим количеством переменных

a = 12
e  = int(str(a) * 3) * 2

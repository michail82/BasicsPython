Значение типа переменных можно менять. Для этого применяются функции:
  • str() - преобразовать в строку;
  • int() - преобразовать в целое число;
  • float() - преобразовать в вещественное число (дробное).  
Иными словами, переменной можно присвоить значение, равное результату выполнения функции над параметром внутри скобок. Разберём это на примерах: 
a1 = 10 # тип целого числа
a2 = 12.17 # тип вещественного числа
a3 = '15.45' # строковый тип
a4 = 'текст' # строковый тип
a5 = '45' # строковый тип

b1 = int(a1) # тип целого числа, результат 10
b2 = int(a2) # тип целого числа, результат 12
b3 = int(a3) # выдаст ОШИБКУ
b4 = int(a4) # выдаст ОШИБКУ
b5 = int(a5) # тип целого числа, результат 45

c1 = float(a1) # тип вещественного числа, результат 10.0
c2 = float(a2) # тип вещественного числа, результат 12.17
c3 = float(a3) # тип вещественного числа, результат 15.45
c4 = float(a4) # выдаст ОШИБКУ
c5 = float(a5) # тип вещественного числа, результат 45.0

d1 = str(a1) # строковый тип, результат '10'
d2 = str(a2) # строковый тип, результат '12.17'
d3 = str(a3) # строковый тип, результат '15.45'
d4 = str(a4) # строковый тип, результат 'текст'
d5 = str(a5) # строковый тип, результат '45'

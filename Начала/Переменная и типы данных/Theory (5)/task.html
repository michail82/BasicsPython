А теперь рассмотрим свойства и особенности этих типов данных, для этого будем складывать и перемножать разные типы.

Отступление. Знак #, поставленный вначале строки, делает всю строку НЕ ВЫПОЛНЯЕМОЙ, то есть Python эту строку не замечает. Если же # стоит не в начале строки, то весь код правее него становится комментарием. Таким образом можно создавать комментарии по ходу кода программы, или быстро исключать строки кода. Запомним: # - знак комментария.

Свойства типов: 
a1 = 10  # это целое число
a2 = 10.0  # это вещественное (дробное) число
a3 = '10'  # это строка (текст), так как в кавычках

b1 = a1 * 2  # b1 имеет тип целого числа и равно 20
b2 = a2 * 2  # b2 имеет тип вещественного числа и равно 20.0
b3 = a3 * 2  # b3 имеет строковый тип и равно '1010'. Строка повторилась 2 раза

c1 = a1 * a1  # c1 имеет тип целого числа и равно 100
c2 = a2 * a2  # c2 имеет тип вещественного числа и равно 100.0
c3 = a3 * a3  # выдаст ОШИБКУ! Нельзя перемножать строки между собой (но складывать МОЖНО!)

d1 = a1 * a2  # d1 имеет тип вещественного числа и равно 100.0
d2 = a1 * a3  # d2 имеет строковый тип и равно '10101010101010101010'. Строка повторилась a1 раз
d3 = a2 * a3  # выдаст ОШИБКУ! Нельзя перемножать строку и вещественное число

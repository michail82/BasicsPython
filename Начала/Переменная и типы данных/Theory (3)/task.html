Для работы с переменной она должна быть явно указана. Как уже было сказано ранее, выполняется это с помощью оператора присваивания, или попросту знака "=".

a = 10


Переменная в процессе выполнения программы может поменять своё значение.

a = a + 1

Таким образом, новое значение переменной (указано с левой стороны) изменилось по сравнению с тем значением, которое было с правой стороны. Конкретно в этом случае, новое значение переменной a стало больше старого значения на единицу. И это новое значение сохранится в программе до тех пор, пока не будет вызвано другое (новое) изменение значения данной переменной.
